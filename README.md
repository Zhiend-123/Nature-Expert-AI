# 🌿 Nature Expert AI
Интеллектуальный помощник натуралиста на базе Google Gemini 1.5

Nature Expert AI — это мощное веб-приложение для мгновенной идентификации объектов живой природы. Просто загрузите фото, и искусственный интеллект определит вид птицы, тип гриба или название минерала, предоставив подробную научную справку.

# ✨ Ключевые функции
🔍 Глубокий анализ: Идентификация видов с помощью компьютерного зрения нового поколения.

🌍 Геолокация (EXIF): Если в фото вшиты координаты, приложение автоматически поставит маркер на карте.

💾 Локальная БД: Ваша история исследований сохраняется в файл и доступна после перезагрузки.

📂 Менеджер истории: Просматривайте прошлые находки, скачивайте текстовые отчеты или удаляйте ненужные записи.

🎨 Современный UI: Чистый интерфейс с темной темой и удобными вкладками.

# 🛠 Архитектура проекта
Приложение построено по модульному принципу для удобства доработки:

📂 assets/ — Кастомные CSS стили для интерфейса.

📂 src/ — Ядро приложения:

brain.py — Логика нейросети.

history_manager.py — Работа с базой данных и GPS.

📄 app.py — Точка входа и пользовательский интерфейс.
